🛒 Retail Sales Data Analytics Project

📌 Project Overview

This project analyzes retail sales transaction data using SQL to understand customer behavior and sales performance.

🎯 Objectives

- Calculate total sales revenue
- Identify top customers
- Analyze customer spending patterns
- Study transaction frequency
- Generate useful business insights

🛠 Tools and Technologies

- MySQL
- SQL
- MySQL Workbench
- Microsoft Excel

📊 Dataset

The dataset contains:

- Customer_ID
- Transaction_Date
- Transaction_Amount

⚙️ Methodology

1. Data Collection
2. Data Cleaning
3. Database Creation
4. Data Import
5. SQL Query Execution
6. Result Analysis
7. Data Visualization

📈 Analysis Performed

- Total Sales
- Total Transactions
- Customer Spending
- Top Customers
- Average Transaction Value
- Monthly Sales
- High Value Customers

📊 Results

The project helped identify:

- Customer purchasing behavior
- Sales trends
- High-value customers

💡 Insights

- Few customers contribute more revenue
- Spending patterns vary
- Sales trends change over time

🚀 Future Enhancements

- Use larger datasets
- Build Power BI dashboards
- Apply machine learning models
